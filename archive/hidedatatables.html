The hideEmptyCols plugin allows you to target columns three different ways. You can use the column index, the column name, or you can specify a negative integer, which would start targeting columns from the right side of the table. This means if you were to use -2 on a table, then you would be targeting the column thats second from the right.

In the example below, were targeting three columns, using two of the methods described. We target the column indexes 0 and 3, as well as the column on the end of the table via -1. The result is an array targeting the Name, Age and Notes columns. Notice that even though those 3 columns were targeted, only the Age and Notes were hidden, because the Name column is populated with values.
